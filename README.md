# Desafío 2 - Condiciones

Proyecto desarrollado para resolver el desafío **“Condiciones”**, correspondiente a la práctica de manipulación del DOM, uso de estructuras condicionales `if`, `else if`, `else` y operadores lógicos con **JavaScript vanilla**.

El proyecto implementa tres problemas independientes solicitados en el enunciado, organizados con una página principal `index.html` y un archivo HTML separado para cada ejercicio.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript vanilla
- SVG propios como assets visuales
- Git para control de versiones

No se utilizan frameworks, librerías externas ni dependencias adicionales.

---

## Objetivo del proyecto

Resolver tres ejercicios de condiciones y manipulación del DOM:

1. Agregar y quitar un borde rojo a una imagen al hacer click.
2. Validar la cantidad total de stickers seleccionados por el usuario.
3. Validar un password usando tres select con dígitos del 1 al 9.
